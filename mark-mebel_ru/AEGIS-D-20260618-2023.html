HIGH Server-Side Request Forgery: callback Parameter L2 CVSS 7.5 CWE-918

Техническая часть

Параметр callback принимает произвольный URL и выполняет запрос с сервера. Тип: internal_access. Техника: Direct: AWS IMDSv1.

Сервер выполняет запросы к внутренним ресурсам в обход файервола.

Бизнес-риск

Доступ к внутренним сервисам (БД, админки, API) в обход файервола. Возможна утечка данных и дальнейшее развитие атаки.

HIGH Rate Limiting Not Configured — Brute-Force Protection Missing L2 CVSS 7.3 CWE-307

MEDIUM Cookie Security: Missing Flags (5 issues in 2 cookies) L2 CVSS 5.0 CWE-614

Техническая часть

Установленные cookie не имеют обязательных флагов безопасности:

  • HttpOnly — cookie недоступна через JavaScript (document.cookie). Без этого флага XSS-атака позволяет украсть сессию пользователя.
  • Secure — cookie передаётся только по HTTPS. Без этого флага cookie может быть перехвачена при MITM-атаке.
  • SameSite — cookie не отправляется при кросс-доменных запросах. Без этого флага возможна CSRF-атака.

Бизнес-риск

Отсутствие флагов безопасности на сессионных cookie позволяет злоумышленнику украсть сессию пользователя через XSS или MITM-атаку, получить доступ к личному кабинету и данным клиента. Это прямой путь к утечке персональных данных и штрафу РКН.
